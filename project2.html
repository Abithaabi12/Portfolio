Library Management System

A backend-based Library Management System built using Java, JDBC, and MySQL. It manages books, members, and transactions efficiently with structured CRUD operations and secure data handling.

Key Features:

  • Add, update, and delete book records
  • Manage member information and issue/return books
  • Real-time database connectivity using JDBC
  • Secure and efficient CRUD operations
  • Simple console-based interface for easy interaction
